Dependencies
------------
1) c++11 compiler.
2) Boost c++ library
3) Python 2.7.12+
4) Linux OS.


Tesing configuration
--------------------
OS: SLES 11SP4
Compiler: GCC 4.9.3
Boost c++ library: 1.60


Building instructions
---------------------
1) To build PBS pro version, type make
2) To build SLURM version, modify Makefile, add -DSLURM to CPPFLAGS, type make.
